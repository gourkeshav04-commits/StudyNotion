# рџ“Њ About the Project

StudyNotion by Apex is a frontend learning platform inspired by StudyNotion.

It allows users to browse courses, explore course descriptions, and navigate through different pages like login, signup, dashboard, about, and contact.

The project demonstrates important frontend development concepts such as:

- Component-based architecture
- State management with React
- Dynamic rendering of course data
- Routing using React Router
- Responsive UI using Tailwind CSS

---

# рџЊџ Features

- Responsive modern UI built with Tailwind CSS  
- Dynamic course listing with course details  
- Read More / Show Less functionality  
- Remove course feature with state updates  
- Login and Signup pages  
- Google Sign-in flow pages  
- Dashboard page navigation  
- React Router based navigation  
- Dark themed modern UI

# рџЋЇ Key Functionalities

### Dynamic Course Rendering
Courses are rendered dynamically using React's `map()` function.

### State Management
React `useState` is used to manage course data and authentication state.

### Navigation
Navigation between pages is implemented using React Router.

### Page Refresh Handling
A `useEffect` hook ensures the application redirects to the home page when refreshed.
